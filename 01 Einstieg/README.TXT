Aufgabe: Kaffemaschine / Vollautomat

Erstellen Sie Programm, welches einen Kaffeevollautomaten 
nachbildet. Er soll zwei bis drei verschiedene Varianten 
zur Auswahl anbieten (z.B. Espresso, Kaffee, Cappuccino).

Schritte
1. Erstellen Sie auf Papier ein UML-Klassen-Diagramm mit den
   erforderlichen Daten und Funktionen
2. Übertragen Sie das Diagramm auf eine neue Klasse und
   bauen das vollständige Gerüst aus Daten und Funktionen
3. Vervollständigen Sie das Gerüst mit dem notwendigen Code,
   so dass Ihr Programm die o.g. Funktionen sinnvoll erfüllen
   kann.
